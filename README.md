# ExpenseTracking
Dự án này là dự án cá nhân của Đặng Vy, sử dụng HTML CSS với Frontend là framework Angular. 
Trước khi bắt đầu dự án, cần phải tải thư viện framwork của Angular về bằng câu lệnh trong cmd.
Tải NodeJS về và chạy câu lệnh:
'npm install -g @angular/cli@19'
Có thể gọi phiên bản tải bằng cách thêm @ cộng số phiên bản ra sau.
Sau đó tạo mới dự án bằng câu lệnh:
'ng new <tên dự án của bạn>'
Rồi 'cd <tên dự án>'
Câu lệnh thêm thư viện antdesign vào
'ng add ng-zorro-antd'
